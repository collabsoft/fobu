FoBu (Form Builder)
===================

1) Quick Start
--------------

Install Node.js:

```
$ sudo apt-get update
$ sudo apt-get install -y python-software-properties python g++ make
$ sudo add-apt-repository -y ppa:chris-lea/node.js
$ sudo apt-get update
$ sudo apt-get install nodejs
```

Install the dependencies:

```
$ npm install
$ bower install
```

Run it:

```
$ grunt watch
```
